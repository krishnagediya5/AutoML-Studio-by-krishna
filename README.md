

# 🚀 **AutoML Studio**

### 🔥 Intelligent AutoML Platform for End-to-End Machine Learning

**AutoML Studio** is a fully automated machine learning platform designed to simplify and accelerate the complete ML workflow. Built using Streamlit and Scikit-learn, it enables users to seamlessly transform raw data into high-performing predictive models through an intuitive, no-code interface.

The platform integrates data preprocessing, feature engineering, model training, evaluation, and deployment into a single unified dashboard — making machine learning accessible to both beginners and professionals.

---

## ✨ Key Features

* 📂 **Seamless Dataset Upload** – Supports CSV files with instant preview
* 🧹 **Automated Data Preprocessing** – Handles missing values, encoding, and scaling
* 🎯 **Feature Selection Engine** – Selects top features using statistical methods
* 🤖 **Multi-Model Training** – Compares multiple ML algorithms automatically
* 🏆 **Smart Model Selection** – Identifies the best-performing model based on metrics
* 📊 **Interactive Visualizations** – Model comparison charts and evaluation plots
* 🔮 **Real-Time Predictions** – User-friendly interface for custom predictions
* 📈 **Dual Support** – Works for both Classification and Regression tasks

---

## 🧠 Core Capabilities

* End-to-end AutoML pipeline
* Automated model benchmarking
* Performance evaluation (Accuracy, F1, RMSE, R², etc.)
* Feature importance visualization
* Model export for deployment

---

## 🛠 Technology Stack

* **Programming Language:** Python
* **Frontend/UI:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Machine Learning:** Scikit-learn
* **Visualization:** Plotly

## 🎯 Use Cases

* Rapid prototyping of ML models
